# gotchi

![gotchi terminal UI](img/pet.jpeg)

`gotchi` is a lightweight terminal virtual pet for pubnix-style communities such as runv.club.

Each Unix user gets an independent pet with persistent state, real-time progression, and a friendly command-line experience. It is designed to stay simple, fun, and low-maintenance: no resident daemon, no web app, just a small CLI companion that lives in the terminal.
